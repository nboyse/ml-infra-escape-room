# ML Escape Room
## Scale, Secure, and Survive

**Scenario**: You just got an emergency call at 3 AM. A new Bedrock-powered GenAI feature is live and on fire.

**Goal**: Fix infrastructure issues across permissions, scaling, load, cost, and production safety before the service dies or the bill explodes.

## Quick Start

```bash
# 1. Clone & setup
git clone <your-repo-url>
cd ml-infra-escape-room

# 2. Start services
docker-compose up -d --build

# 3. Install Locust
pip install -r requirements.txt

# 4. Open browser
open http://localhost:8000
```

Start with challenges/01-permissions-panic/README.md

Teams of 2-4 recommended. First team to finish all challenges wins.
